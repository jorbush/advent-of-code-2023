# Advent of Code 2023

Doing the [Advent of Code](https://adventofcode.com/2023) daily challenges.

## Run the code 

The exercises are done with Python or Rust.

### Run Python exercises

For example, for day 1:

```sh
cd day-1
python3 day1.py
```

### Run Rust exercises

For example, for day 6:
```sh
cd day-6
cargo run 
```
